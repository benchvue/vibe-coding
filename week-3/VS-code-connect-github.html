Connect GitHub from VS Code

Link your local VS Code workspace to a GitHub repository — push, pull, and commit without ever leaving the editor.

1

Make sure Git and a GitHub account are ready

Before connecting, confirm the two prerequisites are installed and that you have a GitHub account:

Install Git   Create a GitHub account

Open a terminal in VS Code with Ctrl + ` (or ⌃ + ` on Mac) and verify Git is available:

$ git --version # e.g. git version 2.43.0
2

Tell Git who you are

Set your name and email — these show up on every commit. Use the same email as your GitHub account so commits get linked to your profile.

$ git config --global user.name "Your Name" $ git config --global user.email "you@example.com"
Tip: Run git config --global --list to double-check your values.
3

Sign in to GitHub inside VS Code

VS Code has built-in GitHub auth — no tokens to manage manually.

Open the Command Palette with Ctrl + Shift + P (or ⌘ + Shift + P on Mac), then type:

> Sign in to GitHub

A browser window opens. Click Authorize Visual Studio Code, then return to VS Code — you'll see your GitHub avatar in the lower-left corner of the window.

4

Pick your path: new repo or existing repo

There are two common starting points. Choose whichever matches your situation:

A. Start from a local folder

You already have files on disk and want to publish them.

Open the folder in VS Code → click the Source Control icon in the left sidebar → Publish to GitHub.

B. Clone an existing GitHub repo

You want to work on a repo that already lives on GitHub.

Command Palette → > Git: Clone → paste the repo URL → pick a local folder.

5

Make your first commit

Edit any file in the folder. In the Source Control panel you'll see the changed file appear.

Hover over the file → click + to stage it, type a commit message at the top, then click the ✓ Commit button.

Shortcut: if nothing is staged yet, VS Code will ask whether to stage all changes automatically — say yes.
6

Push to GitHub

After committing, click the Sync Changes button (the circular arrows icon, also visible in the blue status bar at the bottom).

The first push uploads your branch to GitHub. Refresh your repo page in the browser — your files are there.

# Equivalent terminal commands, if you prefer CLI: $ git add . $ git commit -m "first commit" $ git push
✓

You're connected!

Every future edit follows the same loop: change → stage → commit → sync — all from inside VS Code.
